## 実行方法

必ず以下のいずれかの方法でローカルサーバーを立ち上げて実行してください。

### Node.jsがインストールされている場合:

`npx serve` などを実行。

### VS Codeを使用している場合:

拡張機能「Live Server」をインストールし、`index.html` を右クリックして `Open with Live Server` を選択。

### Pythonがインストールされている場合:

プロジェクトフォルダでコマンド `python -m http.server` を実行し、http://localhost:8000 にアクセス。

## 2. 実験の流れ

実験は 「Phase 1」 と 「Phase 2」 の2部構成で行われます。全30問です。

### 音量調整 (Setup)

実験ページを開くと、最初に音量調整画面が表示されます。

音声を聞き、PCの音量を調整してください。

調整後、「Next」を押して実験を開始します。

### Phase 1: Blind Test (15問)

視覚的な補助なしで、自分の耳だけを頼りにハウリングを聞き分けます。

画面には波形などのヒントは表示されません。

### Phase 2: Assisted Test (15問)

Phase 1と同じ問題が出題されますが、今回は**「支援ツール（波形グラフ）」**が表示されます。

グラフの動きや色の変化（赤く強調される部分など）を参考にしながら回答してください。

データ提出

全問終了すると、結果データが表示されます。

「Copy to Clipboard」ボタンを押してデータをコピーし、指定の送付先（Googleフォーム）に貼り付けて送信してください。

## 3. 操作方法

各問題は以下の手順で進みます。

### Step 1: 検知（スペースキー）

「Start」ボタンを押すと音声が再生されます（ノイズの中に、徐々に「キーン」というハウリング音が混ざります）。

「ハウリングが鳴り始めた！」 と思った瞬間に、キーボードの [スペースキー] を押してください。

※ まだ鳴っていないのに押してしまうと「お手付き（Too Early）」となり、その問題は失敗（記録なし）となります。慎重かつ迅速に判断してください。

### Step 2: 特定（マウスクリック）

スペースキーを押すと音声が止まり、画面にフェーダー（周波数帯域の選択肢）が表示されます。

鳴っていた音の高さ（周波数） だと思う場所をクリックして選択してください。

左側ほど低い音、右側ほど高い音です。

### Step 3: 次へ

「Next Question」ボタンを押して次の問題へ進みます。

※ 学習効果（慣れ）を防ぐため、正解・不正解の結果は表示されません。「Recorded（記録されました）」とだけ表示されますので、気にせず次へ進んでください。

## 4. 注意事項

ブラウザの「戻る」ボタンや「再読み込み（リロード）」を行わないでください。

進捗データがリセットされ、最初からやり直しになってしまいます。

音量にご注意ください。

実験には不快な音（ハウリング音）が含まれます。耳を痛めないよう、最初の調整画面で大きすぎない適切な音量に設定してください。気分が悪くなった場合は直ちに中止してください。
